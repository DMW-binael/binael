LES MAISONS D'ANGES

Au-delà des hiérarchies • Les familles spirituelles

Au-delà des neuf hiérarchies, les anges sont organisés en "maisons" selon leur essence et leur mission. Chaque maison possède ses qualités, ses couleurs, et son rôle dans le grand œuvre cosmique.

☀️

MAISON LUMIÈRE

"Lux in tenebris lucet" • La lumière brille dans les ténèbres

Couleurs : Or, blanc, jaune éclatant

Élément : Feu, lumière pure

Qualités : Clarté, vérité, révélation, purification

Anges : Séraphins, Chérubins, Michel

Mission : Illuminer, révéler, purifier les ténèbres

Symbole : Soleil rayonnant

MAISON SAGESSE

"Scientia potentia est" • La connaissance est pouvoir

Couleurs : Bleu profond, indigo, violet

Élément : Air, éther

Qualités : Connaissance, intuition, mystère, compréhension

Anges : Chérubins, Trônes, Uriel

Mission : Enseigner, guider, comprendre les mystères

Symbole : Livre ouvert, œil qui voit tout

SACRED GUARDIAN THE WATCHER

MAISON PROTECTION

"Custos fidelis" • Gardien fidèle

Couleurs : Argent, blanc-bleu, perle

Élément : Eau

Qualités : Compassion, garde, défense, guérison

Anges : Anges gardiens, Principautés, Gabriel

Mission : Protéger, veiller, guérir les âmes

Symbole : Bouclier, ailes enveloppantes

MAISON TRANSFORMATION

"Per aspera ad astra" • Par les épreuves vers les étoiles

Couleurs : Rouge, noir-or, pourpre

Élément : Feu, terre

Qualités : Courage, changement, alchimie, renaissance

Anges : Anges déchus (repentis), Vertus, Raphaël

Mission : Transformer, transmuter, faire renaître

Symbole : Phénix, ouroboros
